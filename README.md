# gps-tracker
Проект в рамках Embedded Internship в Ajax Systems
Виконавець: Македонський Олександр Дмитрович

Короткий опис проекту: основна мета пристрою - трекінг маршруту за допомогою супутникової GPS-навігації.
Дозволяє окрім безпосередньо самого маршруту збирати дані про швидкість руху, в який час прокладався маршрут ітд.
Працює від заряду li-pol акумулятора.
